# Contributing to Mosaic Protocol

Thank you for considering contributing!

## Contributor License Agreement (CLA)

By contributing to this project, you agree that:

1. **Your contributions will be licensed under AGPL-3.0**
2. **Mosaic Protocol has the right to offer these contributions under alternative licenses** (dual licensing)
3. You have the legal right to contribute this code
4. Your contribution is your original work

### Why We Need This

This agreement allows us to:
- Offer commercial licenses to businesses
- Ensure project sustainability
- Continue developing open-source software
- Protect all contributors

### How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing`)
3. Commit your changes with descriptive messages
4. Push to your fork (`git push origin feature/amazing`)
5. Open a Pull Request

By opening a PR, you acknowledge the CLA above.

## Code of Conduct

- Be respectful and inclusive
- Focus on constructive feedback
- Follow project coding standards
- Write tests for new features
